Principles

Habit-first UX
Daily actions surface one tap away. Pause, resume, track — zero friction.
Quiet surfaces
Soft neutrals, decisive ink. Color is reserved for meaning, not decoration.
Numeric clarity
Prices, quantities and schedules use tabular weight so amounts scan fast.
Progressive disclosure
Cards expand to detail. Lists stay short. Complexity hides behind intent.
